    FourBitNumber
    Remarks
    Channel is a zero-based number in DryWetMIDI, valid values are from 0 to 15. Other libraries and software can use one-based channel numbers (i.e.from 1 to 16) so be aware about that: channel 10 in such software will be 9 in DryWetMIDI.
